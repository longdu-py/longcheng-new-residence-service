# 龙城新居小区便民服务网站
> 纯静态小区便民系统，基于 Bootstrap5 开发，支持 GitHub Pages 一键部署，无后端、开箱即用
## 项目简介
龙城新居业主一站式便民平台，集成物业报修、小区公告、失物招领、邻里团购、停车管理、业主投票、便民商家、投诉建议等全套社区功能，适合本地小区独立使用，完全开源免费。

## 功能列表
1. 首页快捷便民入口、小区实时资讯
2. 物业线上报修+进度查询
3. 小区官方公告分类展示
4. 失物招领/寻物启事发布
5. 邻里生鲜团购板块
6. 周边便民商家黄页
7. 小区车位租赁、停车规范查询
8. 业主线上投票功能
9. 投诉建议反馈系统
10. 快递代收点信息查询
11. 社区志愿者招募
12. 业主个人中心
13. 便民工具：垃圾分类、应急电话、缴费指引

## 部署教程（GitHub Pages）
1. Fork 本仓库 / 新建仓库上传全部代码
2. 仓库 Settings -> Pages
3. Source 选择 `main` 分支，文件夹选择 `/root`
4. 保存，等待1分钟生成网站链接

## 本地运行
直接双击 `index.html` 即可打开，无需服务器。

## 技术栈
- HTML5 + CSS3
- Bootstrap 5
- JavaScript 原生交互
- 本地JSON模拟数据
- 响应式：适配手机/电脑
